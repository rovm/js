# cli-template

## Build Setup

``` bash
# Generate project in current directory?
Y(현재 디렉터리에 프로젝트 생성)

# Project name
cli-template(프로젝트 이름)

# Project description
vue project(프로젝트 설명)

#Author
jsg(만든이)

#License
MIT(라이센스 유형)

#Use sass
N(sass 사용여부)
```

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
